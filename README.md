# 代码说明
1. commonDrawingExamples.ipynb:常用的绘图元素总结，参考[链接](https://matplotlib.org/gallery/index.html)
    - 线形图(包含散点): `ax.plot()`, `ax.scatter()`, `ax.vlines`
    - 柱状图:`ax.bar()`
    - 条状图:`ax.barh()`
    - 直方图:`ax.hist()`
    - 箱线图:`ax.boxplot()`
    - 饼图:`ax.pie()`
    - 热图:`ax.imshow()`
    - 堆叠图:`ax.stackplot()`
